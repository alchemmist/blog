Engineering Open Source Projects

Practice-driven course, was designed by Anton Grishin, where students build a system to evaluate developer contributions based on GitHub activity, implemented as a Python library, a CLI, and a Telegram bot. Students practice modular design, testing, CI/CD, documentation, and real-world Open Source workflows, focusing on usability and clean, extensible solutions. The lectures are given in Russian, but all materials are available in English. Russia, Moscow, Central University campus.
